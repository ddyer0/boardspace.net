NNGS clone for Go Programs


While this server is primarily for people to play games with each other,  Go is a long term interest of mine and supporting the Go programming community is also of interest to me.  I've set up a clone of the No Name Go Server (NNGS) specifically intended so Go programs can play each orther, or play humans if the developers want to.   

  • Access the server at boardspace.net port 6969.  All the standard igs/nngs clients should work.
  • The games played on the server can be retrieved here
