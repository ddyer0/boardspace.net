Loa Tournament status

The first Email LOA tournament is over. The next tournament is tentatively scheduled to start around November 1, 1997. If you might be interested in playing, send a message to ddyer@real-me.net

The tournament was a seven player round robin, played on Richard's email server, starting on October 24, 1996 and ending on February 18, 1997.

The games and commentaries
The games have been commented, mostly by me (ddyer). The skill levels of the players varied widely, which made a interesting mix of games - some blowouts, some tightly contested battles. In any case, with the comments these games are an excellent educational resource.
The games themselves are in a "sgf format" archive , which is somewhat human readable. You can download the game records for offline study, with or without the Java viewer.
