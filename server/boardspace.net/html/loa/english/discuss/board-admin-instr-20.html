Import/Export Utility DISCUS Administration Instructions

The Import/Export utility allows the superuser to import and export the Discus database in several formats.

Export in tab-delimited format: The Discus database will be exported in a tab-delimited format into the file you select. The file will be created in the "backups" directory of your Discus administration directory. You can access this file for download through the Backup Manager interface.

Export messages in offline-viewing format: The Discus database will be exported in HTML format into the file you select. These files are suitable for inclusion on a CD-ROM or browsing through offline viewing. The file will be created in the "backups" directory of your Discus administration directory. You can access this file for download through the Backup Manager interface.
