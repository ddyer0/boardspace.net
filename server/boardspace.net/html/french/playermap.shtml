Carte du monde des visiteurs

La carte ci-dessous indique la localisation des joueurs s'étant connectés récemment, d'après leurs adresses IP. Voici comment ça fonctionne :

  • Le travail de base pour cette carte est de récupérer les adresses IP et leur localisation géographique. Le projet NetWorldMap disponible gratuitement, collecte ces données et permet de les utiliser. Parmi les services gratuits, une API permet aux webmestres de déterminer la latitude et la longitude correspondant à une adresse IP.

  • Losqu'un joueur se connecte, le processus de connexion demande à NetWorldMap sa localisation, et si elle est connue avec assez de certitude, met à jour la base de données avec la "dernière localisation connue" du joueur. Lattitude et longitude sont également envoyées au lobby, qui peut iainsi afficher en temps réel la localisation des joueurs connectés.

  • Le visualiseur de carte est une petite application Java, qui interroge la base de données pour obtenir les dernières localisations des joueurs ayant joué récemment. L'applet digère les données et trace des cercles dont le rayon dépend du nombre de joueurs dans une zone donnée. Bien sûr ce processus n'est satisfaisant que si le nombre de données est assez petit.

  • Une fois que vous avez les outils de base, il y a des tas de choses que vous pouvez traîter avec des cartes. Si vous avez des idées , contactez-moi, j'aimerais beaucoup réaliser des idées à la fois amusantes et utiles. Dave Dyer.
