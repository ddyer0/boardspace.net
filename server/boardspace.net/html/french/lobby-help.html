Guide d'utilisation


Parler à un joueur
 Cliquez sur le nom d'un joueur dans la liste.  Vous pouvez inviter un joueur à rejoindre une partie ou, si un joueur vous ennuie, interdire les invitations de ce joueur, ou ignorer complètement ce joueur.
Débuter une partie
 Un clic suffit !
Entrer dans une salle
 Cliquer sur l'un des rectangles vides pour entrer dans une salle en tant que joueur.  Dans les parties entre humains, le rectangle du haut désigne le premier joueur, celui du bas le second.
Choisir un type de salle
 Choisissez parmi les types disponibles (en cliquant sur le type d'une salle) - Salle de jeu, Salle de maîtres, Salle d'entrainement, ou divers types spécialisés.  Les salles d'entrainement permettent des parties sans classement.  Les salles de maîtres ne sont accessibles qu'aux maîtres.  Les salles d'étude permettent de revoir des parties archivées.  Les salles de dialogue sont là pour dialoguer !  Si vous envisagez une longue discussion, choisissez d'entrer dans une salle de dialogue plutôt que d'ennuyer tous les joueurs présents.
Sélectionner un jeu
 Dans une salle vide, vous pouvez sélectionner le jeu que vous désirez (en cliquant sur le nom du jeu).   Zertz, Hex, Trax, Punct, Yinsh, Gobblet, Hive ... et tous les autres.
Paramètres du jeu
 Il y a trois choix, "Rejoindre" permet à tout joueur de se joindre à la partie.  "Rejoindre sur invitation" signifie que les joueurs ne doivent se joindre à la partie que s'ils y sont invités. Seul le premier joueur dans une salle peut débuter la partie.   "Tournoi" signifie qu'il s'agit d'une partie d'une tournoi en cours. Les parties de tournoi seront nommées"T!quelquechose" dans les salles d'étude.


Options
 Vous pouvez couper le son ou les dialogues, interdire toute invitation à jouer.
Temps GMT
 Utiliser l'horloge GMT est le moyen le plus simple de s'accorder sur une heure pour disputer une partie de tournoi.

Autres caractéristiques

  • Vous pouvez fermer cette fenêtre sans affecter les parties en cours, mais votre browser doit toujours tenir ouverte la page principale du site.
  • Les parties classées des robots sont enregistrées sur le serveur. Vous pouvez donc quitter ces parties puis les poursuivre plus tard. Dans ce cas, le bouton disant normalement "jouer contre le robot" dira plutôt "rejoindre le robot".
  • Vous pouvez reprendre votre partie contre un humain après un plantage sur votre ordinateur, tant que votre adversaire ou au moins l'un des spectateurs de la partie reste dans la salle.  Si le plantage affecte l'ensemble du site, vous n'avez pas de chance !  Si vous pouvez rejoindre la partie, le bouton disant normalement "observer" dira plutôt "rejoindre".
