Tzaar

Tzaar est un jeu associant empilement et capture. Chaque tour consiste en deux déplacements ; le premier doit entraîner une prise, le second peut être une prise ou un coup de défense. Vous commencez la partie avec une quantité différente de trois types de pièces. Elles ont toutes les mêmes capacités, mais il faut garder les trois types sur le plateau. Globalement, il est nécessaire de garder un bon équilibre entre attaque et défense.

Lisez les règles officielles sur le site Internet du projet Gipf.

Robots : Les joueurs expérimentés ne se sont pas encore prononcés, mais le robot est suffisamment adroit pour tenir les débutants en haleine.
