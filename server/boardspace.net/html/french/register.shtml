Inscription


S'inscrire et jouer sur Boardspace est gratuit.  Vous êtes cordialement invités sur ce site, merci d'y respecter les règles de bienséance. Bonnes parties !
Nom
Votre véritable nom, ou autre chose !
Boardspace.net utilise Java.  Avant de pouvoir jouer, une plateforme java doit être installée pour votre explorateur. Voir la page NeedsJava pour quelques conseils.
Pseudo
Votre pseudo, 3 caractères au moins, lettres et chiffres autorisés.
mot de passe
 Choisissez un mot de passe ni trop simple, ni trop complexe.
confirmer le mot de passe
Comfirmez le mot de passe. Un mot de passe perdu peut être récupéré, si votre adresse e-mail est valide.
email
Vous recevrez un courriel pour activer votre compte, l'adresse que vous saisissez doit donc être valide.  Elle ne sera pas utilisée en dehors des communications officielles émanant du site (pour un mot de passe oublié par exemple) et ne sera ni vendue ni cédée à une tierce partie.  Pas de spam ici.
Pays
Pour la carte des joueurs, les tournois et ce genre de choses. Si vous avez une âme d'apatride choisissez "Planète Terre".
