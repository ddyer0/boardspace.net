Tako Judo

"Tako" is how the Japanese word for Octopus is written in English.  so "Tako" is octopus, and "Tako Judo" is octopus wrestling.

If you ignore the silly theme, Tako Judo is an abstract positional game with very simple rules.  The object of the game is to immobilize your opponent's octopus, which has 8 tentacles, which are mobile only when they have a clear line of sight back to their octopus head.  Everything, including the octopus head, moves like a chess queen.   It's a delightfully simple game.

Robots:  Barely adequate.
