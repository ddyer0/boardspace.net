Tumbleweed

Tumbleweed is an area control game, played on a hex-hex board, where line of sight from existing pieces is the central feature.
The game starts with a neutral stack in the center of the board, and the first player places one "seed" piece for each color.
The second player chooses which color to play, and subsequent moves are made only to hexes with line of sight to an existing hex of the player's color.  Read the complete rules here.


Robots:  Robot seems to be adequate, if not stellar.
