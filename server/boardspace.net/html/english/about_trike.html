Trike

Trike is a perfectly simple abstract.  Starting with an empty board, place a pawn then move it in a straight line.  When the pawn can't be moved, the game is over.  Read the complete rules here.

Trike has no obvious strategy, but once the number of acessible cells is small enough an exact solution is easily calculated.  Calculable situations are effetively traps for the unwary.

Robots:  Impossible to beat once the endgame is in sight. You must have won before that.
