BugSpiel

BugSpiel is an original game inspired by Wingspan.

The main inspiration is that, just as there are an enormous number of interesting bird species,  there are even more species of  bugs.  Bugspan features a large deck of  ""Bug Cards" which are intended to be reasonably true to life.   The whole deck can be found here.  Corrections welcomed!

The main mechanics in BugSpiel are to buy bugs, deploy them on the board, and score bonuses based on the bugs in the same lines.  There's no hidden information, and quite a bit of opportunity to interact (or if you prefer, interfere) with other players.  The detailed rules are here.

Robots:  Not bad at all, but beatable.

The back story:  Stonemaier games "Wingspan" immediately suggested similar games based on decks of cards.  For me that immediately suggested "BugSpan", but in deference to Stonemaier's "-span" franchise, this game is called BugSpiel.   It languished among my vague plans for a long time because I knew I needed an entomologist as a collaborator.  It turns out, ChatGPT is a pretty good partner!
