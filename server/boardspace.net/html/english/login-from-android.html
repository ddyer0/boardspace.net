Boardspace from Android Phone or Tablet

Windows download    Macintosh download   Linux download   Iphone/Ipad download 

Boardspace has a free app which should run on any Android device.  Search for the Boardspace.net app on Google Play or the Amazon app store.


When you run the app, the first thing you should see is the login screen.  Eventually you will want to create an account (it's free) but
the first time you can check the "guest" box and log in as an anonymous guest.

Note that some games may not play well on smaller screens. Use your own best judgement for that, but bigger screens the better!  Also, for many games, only the weaker robots are recommended to play.   PC desktops are a lot more powerful than mobiles, and can run stronger robots.

A general guide how to use Boardspace after you log in is here.
