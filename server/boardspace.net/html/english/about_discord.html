Boardspace and Discord 

Discord is a free service that provides interactive text,  images, audio and video.  It's good for real time chats, and not bad for blog-like chatter.  It's widely used in the game community.
Apps are available for all the common platforms, and they all share the same content.  Boardspace encourages everyone to set up a discord account, install the apps on your devices.

Boardspace has a discord server which you should join.  Here's the invite.

I recommend using discord
  • as an alternative to Zoom to accompany real time games among friends.  
  • to meet, greet and monitor opponents in tournament games.  
  • for general chatter about the site or playing on it.
One of the channels on the boardspace discord server is the "tournament-notifications" channel., which logs activity in boardspace tournaments.  It's probably too noisy to watch all the time, but if you jump through a few hoops, it's possible to get a realtime notification on all your discord apps whenever a tournament e-mail is sent to you.  The hoops are:
  • find your discord user id (it's an 18 digit number).  It's currently difficult to find on discord but here are the instructions.
  • edit your boardspace profile and add the ID to it.
  • in the boardspace tournament-notifications channel, set the notification-settings to @mentions only
