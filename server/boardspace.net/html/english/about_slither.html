Slither

Slither is a connection game on a square grid, played on any size square board.   White connects left to right, Black connects top to bottom.   The game is driven by the restriction that diagonally adjacent stones of the same color are forbidden.  Read the complete rules here (they're very short!).

There isn't much known about the strategy and tactics.  I found this and this  - two versions of the same
document - which don't cover exactly the same ideas.

Robots:  Robot is very bad.    I'm working on it but not expecting great improvement.
