Here are all the Trax games that have been played at boardspace.net, plus some other games collected from other sources.  The most interesting of these is the large collection of commented games, prepared by Donand Bailey.   To view these commented games, select the commented folder
in the direcotory browser
You can get the same review capabilities in the game lobby by selecting Review Room and Trax,  in any room in the boardspace lobby.  The advantage of using the online version is that you can jointly review games with other players.
