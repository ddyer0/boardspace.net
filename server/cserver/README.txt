
This folder contains the source for the game server that is actually deployed.
use the makefile to build it, copy it to the ../bin directory to be actually
used.  

 to test and use. 

	./boardspaceServer test.conf

Copy to ../server to deploy for actual use. (archive the old version first!)

banner is an obsolete side project to generate per-host keys required to run the server.
error is an unused (lately) side project to provide explanations for unix error numbers.


NNGS is an obsolete side project, the "no name go server" which once
hosted live go games.  I didn't develop it but ran this copy for a
while.
