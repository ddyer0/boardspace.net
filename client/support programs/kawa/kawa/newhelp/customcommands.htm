Select Customize/Commands menu option to setup custom commands or Kawa plugin commands under Kawa.

This will display a dialog as shown below.

Select New to setup a new command. Type the menu name to display for the command.

Select the type of command - Custom or plugin. Please refer to Kawa Plugin for more details on plugin command.

Select the command executable file under the command field. Without this field the command will not be added to the customize menu.

Type in any arguments in the arguments field or pick any of the standard Kawa parameters as arguments.

Please select the appropriate start up directory or the parameter for the command.

Any path argument, passed by the selection of Kawa parameter, will be relative to the startup directory by default. Checking Use absolute paths checkbox will force the paths to be absolute instead.

Checking use project classpath will set the classpath to currently open project classpath prior to running the command.

Move Down and Move Up can be used to rearrange the commands in the listbox to set the order of the commands.

Delete will delete the command from the listbox.

Input/Output redirection will let you redirect the command output and input to different windows. The options are self-explanatory.

Command Options -

Execute command when Kawa starts is valid for Plugin commands only.

Selecting add command to Customize or plugin menu will add the command to the appropriate Kawa top level menu.

Add command to File Item right button menu in the project tree will add this command to the right button menu of a file item in the Project tree

Add command to Project Item right button menu in the project tree will add this command to the right button menu of a project item in the Project tree

Add command to right button in editor menu now allows menus to be specifically added to the context menu of the editor window.

By specifying a file extension for the command, the command will now only be applied to files with the matching extension. This field can only be used if the add to file item or add to editor context menu is also checked. Please note the '.' in front of the extension name and there can be only one extension.
