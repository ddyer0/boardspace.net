Source browsing allows you to find the files where a given name is defined. The name can be a class or a method or a variable. Also, source browsing can search both JDK and the current project. Select View/Source Browse menu option to display the source browsing dialog box as shown below.

  • After typing in the appropriate search string and selecting where to search, click the search button. The results show up in the bottom window, grouped according to type. The bottom window can be resized as necessary, and either double clicking on the item or selecting display will open the corresponding file up in the editor. The source browsing window stays open until you want to close it.
  • Shortcuts can be used to access this dialog. Ctrl + B perform a search based on highlighted text. Ctrl + K will search the project and package list for a class based on highlighted text. Ctrl + M will search the project and package list for a method based on the highlighted text. If there is no text highlighted, the dialog will open setup in the mode based on the keystroke.
