When you license Kawa you will obtain a key and a code to unlock the software. There are 2 places where you can type this key and code to unlock the software. If the product has already expired please click on Register button on the product expired dialog and type the information. If the product has not expired and if you are able to bring up the product please select Register from the Help menu to display the registered dialog as shown above and enter in the key and code.
