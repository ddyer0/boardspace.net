Select Customize/Options/Color... to display the directories dialog box

  • Background color will set the background color options for the editor window .
  • Editor Margin color sets the color for the strip along the left side of the editor window.
  • Text type includes
    • Comment - Comment text
    • Data types - Data types like int, float etc.
    • Keywords - Language keywords like static, class etc.
    • Normal - Regular text
    • Number - Numbers
    • String - Quoted text
  • Font Style will select the appropriate font for the selected text type. It includes.
    • Regular
    • Bold
    • Underline
    • Italics
  • Font Color will select the font color for the selected Text Type
  • Default will select the default color options for all the different text.
