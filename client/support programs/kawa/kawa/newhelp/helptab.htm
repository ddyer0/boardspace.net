  • Help tab is a customizable help tree for maintaining different kind of documents. Mainly it is used for HTML files but it can be used for Java files, Windows help files and any text files.
  • Double clicking on a file will open the file in the browser if it is a HTML file otherwise it is opened in the Editor for editing
  • Add Folder will add a new folder to the tree
  • Add File(s) will display the File Dialog to add files to a folder.
  • Add URL will let you add URL links to the tree
  • Delete will delete the selected item - a folder or a tree item
  • Edit will open the file in the editor for editing. Same as double clicking for all files except HTML type.
  • Move up and Move down will move the item up or down in the tree.
  • Rename will allow you to set a more descriptive name for the item in the tree but still referring to the same file.

Please refer to How to setup Kawa Help for more information on how to add Kawa help files into this tab.
