Bookmarks as the name suggests is a marker within in the editor window. Once a book mark is set in the editor you can jump quickly between bookmarks by using the keyboard shortcut SHIFT+F2. Bookmarks under Kawa are temporary. This means when a file editor is closed all the assosiated bookmarks are lost. A file can have unlimited bookmarks.

Select Edit/Bookmark menu option or the shortcut CTRL+F2 to insert or remove the bookmark at the current line

Select Edit/Next bookmark or the shortcut SHIFT+F2 to move from one bookmark to another within the same file.
