It is not necessary to make a project under Kawa to compile Java files. Follow these steps to compile a single Java file under Kawa

  • Please make sure your Profile paths are set correctly to the desired JDK version.
  • Open the Java file in the editor and select Build/Compile or press F7 to compile the file.
  • If there are any errors in the file the output/build window will list all the errors. If the output/build window is not visible select View/Output menu option or ALT+2 to display the output/build window.
  • Double click on any error line which has a line number to position the error cursor in the editor. You can also use F6 or SHIFT+F6 to move between different error messages.
  • Kawa allows compiler options to be set for single Java files or non-project Java files. Please select Customize/Project Root options to set these options. Setting these options will apply it globally to all compiles.
