Please select Build/RMI compile to invoke the rmic command of JDK. This command will let you create stubs and skeleton classes for remote objects using Java RMI. Please refer to Java RMI documentation for more details on rmic options.

Most options in the above dialog are self-explanatory. Custom options will be passed as is to the rmic command.

Add files will let you add fully qualified class names from the selected java files. Kawa will automatically parse the selected java files and determine the list of classes to add to the list at the bottom of the dialog.

Delete will delete the selected classes from the file list.

Clear All will clear all the options and the classes list.

Start will start the rmic command with all the options specified in the dialog and the output from the rmic command can be seen in the output window just like the compiler output.

RMIC options from the dialog are saved with Kawa project automatically.
