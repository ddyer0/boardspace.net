This document discusses setting up the basic JDK paths and Kawa paths. Please refer to Advanced concepts for setting the advanced paths. Select Customize/Options/Directories... to display the directories dialog box as shown below -

  • Internet Browser file is the browser program file for the browser. This information is used in some commands to open the HTML file in a browser. This field is optional.
  • Kawa configuration directory is the location of Kawa data files. On a single user installation this location should be the same as the location of the Kawa program file. On a multi user installation this location is the path to all the Kawa data files. Please refer to Kawa multi-user setup for more information on multi user or server based Kawa setup. Kawa configuration directory should be set automatically by Kawa setup but please make sure it always points to the correct location at all times.

Please refer to Profile for more details on profiles.

Java Paths -

  • Java Bin directory is the JDK bin directory. Please select the bin directory under your JDK installation.
  • This directory should be filled automatically by Kawa when you select the bin directory. Please check to make sure it points to the correct location
  • Java Source directory is the location of Java source files directory. This information is used to open the source file during debugging and on double clicks from the package tab. Please refer to setup document for a special note regarding JDK 1.2 with the Java source location.
  • Java documents directory is the location of Java document sub-directory. This information is used to display context sensitive JDK help in the editor.

Kawa Paths -

  • Please refer to advanced concepts for setting the value of Package source directories. This is an optional field.
  • The help index file is the file used by Kawa to load JDK HTML help information when the user presses F1 in Kawa editor. The default index file is Kawa.ipg and should be located in the Kawa directory. Please refer to setup for more information on this field.
  • The package index file is the file used by Kawa to load the package tree with package information. The default file is called Kawa.pkg and should be located in the Kawa directory. Please refer to setup for more information on this field.

Advanced Paths -

    All paths in this tab are set automatically by Kawa. Please read advanced concepts for more information on these paths.
