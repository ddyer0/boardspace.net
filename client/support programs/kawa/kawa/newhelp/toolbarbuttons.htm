Main Toolbar -

  • File New
  • File Open
  • File Save
  • File Save all
  • Cut
  • Copy
  • Paste
  • Quick Find - Type the word to find and press enter or F3 to find the text in the open file.
  • Find Next - Find next instance of the word in the find combo.
  • Find Prev - Find previous instance of the word in the find combo
  • Toggle Project view - Toggles the project view between file view and class view
  • Stop Build
  • Compile
  • Rebuild All
  • Rebuild Dirty
  • Run the app or application if not running. Stop the application if already running.
  • Windows - Display the list of open Windows as a popup menu

Debug Toolbar -

  • Start the Debugger
  • Stop the Debugger
  • Continue
  • Step
  • Step Over
  • Step Up

Browser Toolbar -

  • Display Help/Search Dialog
  • Open a dialog to type a URL name
  • Stop loading the URL if loading
  • Display help history
  • Help back
  • Help Forward

Profile Bar-

  • Current profile selection with a list of all available profiles.
  • Display Options - Displays the Customize/Options Dialog.
