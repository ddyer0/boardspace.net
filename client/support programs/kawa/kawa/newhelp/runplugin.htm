Select Plugin/Run Plugin command to run a single plugin command.

Select the main class of the plugin command and select any arguments or parameters in the parameters field.

Add any classpath required for the command and click on run to run the plugin command.

The startup directory defaults to the location of the class file. Please change it to approriate directory before running the command.
