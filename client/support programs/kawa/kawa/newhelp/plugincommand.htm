Select Customize/Commands... to add a new Plugin command as shown in the figure below.

Please type a menu name for the command. This is the name that will be added to the menu bar.

Please select the class that needs to be run. Please make sure you add any package name that may be required to run the class.

Please add any arguments that need to be passed to the command. Please refer to kawa parameters for passing Kawa parameters as arguments.

Finally, set any special classpath needed to run this command. Kawa automatically adds the classpath to Kawa plugin classes and JDK classes. Please be sure to add any other classpath that may be required.

Please refer to custom commands for more details on the options tab.
