The default JDK class library, classes.zip for JDK 1.1 and rt.jar for JDK 1.2, is not compiled with debug information. Therefore, during debugging if you try and expand a JDK class in Kawa you get the error message that there is no debug information available for this class.

Please download the debug version of classes file from the Kawa site and follow the instructions on the site to install this file.

Please indicate the location of debug class file under Customize/Options/Profile/Advanced paths/JDK Debug classes path. Please make sure the debug class file version and JDK version matches. Kawa will automatically set the classpath for this file during debugging.
