Please read How to create and maintain a Project for more information on creating and maintaining a project. Please follow these steps to compile a project.

  • Please make sure the project is open.
  • Select Project/Rebuild All menu option to rebuild all the files in the project
  • Select Project/Rebuild Dirty menu option to rebuild only those files that need to be compiled. Kawa parses all the project files to find out which file needs to be compiled. This may not always compile the right files. Please use Rebuild All if you want to compile all files.

Please refer to JDK documentation on javac compiler for more information on how the output class file path is determined. If you want to set the output directory for your compiled files please select Project/Compiler options menu to set the compiler output directory and any other compiler options
