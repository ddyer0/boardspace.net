It is quite easy to create packages using Kawa. Please read Java documentation on what is a package prior to reading this document.

Since Kawa uses the underlying JDK compiler to compile all the files the package creation under Kawa is no different than the JDK package creation. It is important to create a Kawa project to build packages.

Specify a package statement as the first line in your source file. Optionally, specify a compiler output directory under Project/Compiler options The compiler will automatically create all the necessary folders for the output files.

Kawa will always add the output directory to your project classpath so the package classes can be found during run and builds.
