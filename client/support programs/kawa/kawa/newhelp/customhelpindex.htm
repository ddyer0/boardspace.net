With Kawa you can create your own help index file. This will provide F1 context sensitive help for your keywords in the editor.

This feature will let you add custom help for other JavaSoft APIs like Java3D which Kawa does not provide.

The help index you create are is displayed as a list of keywords in the Help/Search dialog under the user tab.

Custom indices can be built from any HTML file. If you are building an index from Javadoc generated HTML like all the APIs from Javasoft please select allnames.html to build the index from.

Select Customize/User Index to build a new use help index. This will display a dialog as shown below.

Select Add File... button and select one or more HTML file to scan the keywords for. Depending on the size of the HTML file this could take a few minutes. Please check the progress bar at the bottom of the screen.

When Kawa is finished scanning the HTML file it will display a list of keywords found in the HTML file(s) as shoen below.

Please Delete any indices which you don't need and select Add to add these indices to the User index.

Add will close the second dialog and bring you back to the first dialog. At this time you can either save the newly generated user index or cancel.

If you save the new user index and press F1 on a keyword in the editor that is part of the new user index you just built, Kawa will bring the HTML documentation referred to by the keyword.
