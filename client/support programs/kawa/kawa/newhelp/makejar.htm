Please refer to Jar documentation under JDK for an explanation of the different Jar options. Select Build/Make Jar menu options to display the Make Jar dialog as shown below.

This dialog is a modeless dialog.

  • Make jar dialog is only available for creating a new jar
  • Add File will display file selection dialog to select the files that will be added to tha jar. The filenames will be passed relative to the output file directory.
  • Delete will delete any files selected in the files list control.
  • Clear All will clear all the options from the dialog box.

Selecting Start button will start the jar command and the output from the command is redirected to the Kawa output window.
