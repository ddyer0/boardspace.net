How to override base class methods

We will go back to DrawTest tutorial discussed under tutorial 1 for learning how to debug.

  • Please select Project/Open menu option and select the DrawTest.kawa project file in the DrawTest folder to make it as the current project.
  • Postion your cursor within the body of DrawTest class and right click your mouse button. Select Override menu from the menu
  • You will now see a submenu showing the hierarchy of DrawTest class. This information is read from the package tree.
  • Selecting any class from the submenu will now display all the available methods from that class which can be overridden
  • Note how the methods which have been already overridden are disabled from selection
  • Selecting any method from the methods menu will trigger Kawa to insert a method template inside the body of the class
