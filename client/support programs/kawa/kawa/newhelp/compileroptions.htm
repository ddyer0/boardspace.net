Please refer to JDK documenation of javac for more details on each of these options. Following is a brief description of some important options.

  • Checking compilation output directory is equivalent to -d option value for javac compiler. This is especially important when building packages
  • Checking Debugging tables will build debug information. Required for debugging the application
  • Checking Deprecated API will not display the deprecated API warnings
  • Custom options are sent as typed to the compiler. Please use this option if Kawa does not directly support some compiler option

Compiler options can also be set globally for all the projects by selecting Customize/Project Root options. Setting compiler options globally will set the options for all the projects. However, project compiler setting will supersede the global compiler option.
