Selecting Help/Search menu option will display the following dialog. Each tab indicates a separate help index file.

JDK help index is shipped with Kawa and user index is the index of customized user indices as built by the user.

Please refer to Kawa setup for more details on how to setup JDK help under Kawa.

Please select the tab for the help index you want to search. Find the keyword and double click to display a list of topics available for the keyword.

Double click on one topic to display the HTML help assosiated with the topic.

You can also yype the keyword you are searching for in the list box and Kawa will auto scroll the keywords list.

Pressing F1 in the editor window will automatically search for the keyword at the cursor and display the HTML help if there is only one topic found or the topic list of more than one topic is found.

F1 context sensitive search will search JDK index first and user index next for the keyword match.
