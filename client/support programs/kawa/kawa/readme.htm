Upgrade Note - The new Kawa documentation is managed fully from Kawa help tree. If you upgraded to Kawa 3.2 from an older release then you are not seeing the new Kawa Help tree because the upgrade install does not overwrite your existing kawa.hpg file. In order to see the new Kawa documentation help tree please rename kawadocs.hpg in the kawa folder to kawa.hpg or go to File/Open menu option and open kawadocs.hpg from Kawa folder.
