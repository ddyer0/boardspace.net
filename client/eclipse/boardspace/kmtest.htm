Keyboard/Mouse event testing


________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________
1. The status bar will show all keyboard and mouse event.
2. All keys press/release should be handled. Key code will be displayed on the status bar.
3. All mouse move should be handled. Mouse position will be displayed on the status bar.
4. All mouse button press/release should be handled. Mouse button code will be displayed on the status bar.
5. If you cannot see the correct event status, it means that Ajax VNC is not compatible with your browser.
